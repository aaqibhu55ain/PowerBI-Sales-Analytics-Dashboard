Title:
 Sales Analytics Dashboard (Power BI Project)
 
Project Overview:
 This project presents an interactive Sales Analytics Dashboard built using Microsoft Power BI.
 The dashboard analyzes sales performance, profit trends, regional distribution, and product contributions using the Superstore dataset.

 Dashboard Features:
 • Executive Sales Dashboard
 • Sales vs Profit Scatter Analysis
 • Category Contribution Treemap
 • Regional Sales Performance
 • Interactive filters for Year, Category, and Region

 Key Insights:
 • Office Supplies generates the highest overall sales.
 • Technology category produces the highest profit.
 • Consumer segment contributes the majority of orders.
 • Sales increase toward the end of the year indicating seasonal demand.

Tools Used:
 Power BI
 Visualization
 Business Intelligence
